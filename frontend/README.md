# Vantis Frontend

A modern self-custodial smart wallet application built with Nuxt.js.

## Features

- Home dashboard with portfolio overview
- DeFi integration for funding and token swaps
- Pay Mode with instant payments and installment plans
- Settings and wallet management
- Onboarding flow with passkey authentication

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Pages

- `/` - Home page
- `/select-amount` - Select borrowing amount
- `/defi` - DeFi funding and swap options
- `/pay-mode` - Payment options (instant and installments)
- `/settings` - App settings
- `/wallet-address` - Wallet address display
- `/onboarding` - Passkey setup
- `/welcome` - Welcome/intro page

## Tech Stack

- Nuxt.js 3
- Vue 3
- CSS Variables for theming
- Mobile-first responsive design
